# 🚀 Vinalax HR Solutions - Complete MERN Stack Application

## 📋 Project Overview

**Vinalax HR Solutions** is a **fully-responsive MERN stack application** with SQL Server database for HR compliance and payroll management services. The application features dynamic service pages, WhatsApp-style appointment chatbot, and comprehensive booking management.

---

## ✨ Key Features

✅ **Fully Responsive Design** - Mobile, Tablet, Desktop optimized  
✅ **Dynamic Service Pages** - Click any service card to view detailed information  
✅ **About Us Page** - Company mission, vision, values, and detailed information  
✅ **WhatsApp-Style Chatbot** - Floating widget for instant appointment booking  
✅ **Smart Appointment System** - Select dates, time slots, confirm bookings  
✅ **SQL Server Database** - Secure data storage for all bookings  
✅ **Real-time Slot Management** - Automatic availability tracking  
✅ **Mobile-First Approach** - Optimized for all screen sizes  

---

## 🛠 Tech Stack

| Component | Technology |
|-----------|-----------|
| **Frontend** | React 18, React Router DOM, Axios, CSS3 |
| **Backend** | Node.js, Express.js, MSSQL |
| **Database** | SQL Server 2019+ or Azure SQL |
| **Styling** | Pure CSS (Responsive) |
| **APIs** | RESTful with error handling |

## 🚀 Quick Start

### Step 1: Clone Repository
```bash
git clone https://github.com/niki08fm/vinalax-hr-solutions.git
cd vinalax-hr-solutions
```

### Step 2: Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your SQL Server credentials
npm start
```

### Step 3: Database Setup
Run `backend/config/schema.sql` in SQL Server Management Studio

### Step 4: Frontend Setup
```bash
cd frontend
npm install
echo "REACT_APP_API_URL=http://localhost:5000/api" > .env.local
npm start
```
